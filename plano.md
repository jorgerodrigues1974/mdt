# Plano de Desenvolvimento - MDT Lite 🚀

Este documento serve como o Roadmap oficial para o projeto MDT Lite. Aqui registamos os objetivos atuais, tarefas pendentes e visões para o futuro.

## 🎯 Objetivos Atuais (Q2 2026)
- [x] Estabelecer fluxo de trabalho profissional (Git + Docs).
- [x] Sincronizar primeira app local (ArcGIS Survey123).
- [ ] Implementar sistema de logs de auditoria detalhado.
- [ ] Refinar a interface de utilizador (UI) para um estilo mais premium (Microsoft Fluent Design).

## 📋 Lista de Tarefas (Backlog)

### Alta Prioridade
- [ ] **Segurança**: Migrar segredos e passwords da dashboard para variáveis de ambiente ou ficheiro seguro.
- [ ] **Sincronização**: Melhorar a lógica de sincronização da biblioteca (melhor detecção de versões).

### Média Prioridade
- [ ] **Templates**: Adicionar suporte para templates de instalação (grupos de apps pré-definidos).
- [ ] **Notificações**: Implementar alertas via Email/Slack após conclusões de instalações remotas.

### Futuro / Ideias
- [ ] Dashboard de estatísticas de uso.
- [ ] Suporte para instalações via Chocolatey além do Winget.
